HTML to język znaczników, który służy do tworzenia struktury stron internetowych. Dzięki niemu możemy dodawać nagłówki, paragrafy, obrazy, linki oraz formularze. Każda strona oparta na HTML ma swoją unikalną strukturę.

CSS odpowiada za wygląd strony internetowej. Możemy za jego pomocą zmieniać kolory, czcionki, rozmieszczenie elementów oraz tworzyć responsywny design. Dzięki CSS strona wygląda estetycznie i profesjonalnie.
